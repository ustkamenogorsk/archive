# Костюрин Орест Федорович[^1]

Годы жизни: 1859 – ноябрь 1919-го.

Жена: Герасимова Ольга Георгиевна (Егоровна)

Прибыл в Устькаменогорск в конце 1883 будучи высланным по приговору суда на 5 лет.

Владелец слесарно-кузничной мастерской. Был избран в городскую думу в качестве гласного, а затем и как градской глава.
